
# BoardCalculate - Lumber and Wood Calculators

## Project Description

BoardCalculate provides free lumber and woodworking calculators to help woodworkers and DIY enthusiasts convert board feet, calculate material needs, and plan woodworking projects with precision.

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Running the Project Locally

Follow these steps to run the project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd boardcalculate

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
```
